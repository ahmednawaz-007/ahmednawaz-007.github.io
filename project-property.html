How it works

01
Applicant fills form Tally.so form collects: name, income, move-in date, reason for moving, pets, guarantor info
02
n8n captures instantly Webhook triggers immediately on submission. Data formatted and prepared for AI scoring.
03
AI scores applicant (1-10) Groq API (LLaMA 3.1) analyzes: income vs rent ratio, stability signals, red flags, overall fit. Scores in <1 second.
04
Intelligent routing Score 7-10: Auto email with viewing link | Score 4-6: Request more info | Score 1-3: Polite decline
05
Airtable updates in real-time Property manager sees clean shortlist organized by score tier. Zero manual data entry.
